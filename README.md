# 📊 Personal Glassmorphism Dashboard

Стильная и функциональная персональная панель управления (Dashboard), выполненная в современном дизайне Glassmorphism (эффект матового стекла). Проект полностью автономен, работает на чистом фронтенде без использования сторонних серверов и баз данных.

👉 **[ПОСМОТРЕТЬ ЖИВУЮ ДЕМО-ВЕРСИЮ](https://github.io)**

## 🔥 Основной функционал

*   **Real-time Часы и Календарь:** Отображение точного локального времени, даты и дня недели на русском языке с динамическим обновлением каждую секунду. Автоматическое определение времени суток для вывода приветствия.
*   **Умный виджет погоды:** Автоматическое определение города пользователя по IP-адресу и вывод актуальной температуры и облачности через стабильный текстовый шлюз `wttr.in`. Полная защита от CORS-ошибок.
*   **Генератор мудрости:** Расширенная встроенная база мотивирующих и юмористических цитат для разработчиков на русском языке, переключаемая в один клик без задержек и запросов к сторонним API.
*   **Локальный список задач (To-Do List):** Интерактивная панель задач с возможностью добавления, вычеркивания и удаления. Данные сохраняются в память браузера (`LocalStorage`) и не пропадают после перезагрузки страницы или закрытия браузера.
*   **Фильм дня:** Автономный рандомайзер кинокартин с высоким рейтингом, выдающий название, оценку и краткое описание шедевров мирового кинематографа на русском языке.
*   **Кибер-Зеркало:** Интеграция с веб-камерой устройства через встроенное браузерное API для вывода плавного видеопотока в реальном времени прямо внутрь матовой карточки.

## 🛠️ Стек технологий

*   **HTML5** — семантическая структура, разметка интерфейса и использование встроенных медиа-тегов.
*   **CSS3** — использование Flexbox для адаптивного расположения элементов в одну строку, кастомные градиенты, размытие фона (`backdrop-filter`) и плавные анимации при наведении.
*   **Vanilla JavaScript (ES6+)** — асинхронные запросы (`async/await fetch`), работа со временем (`Date`), потоковый захват медиа (`navigator.mediaDevices`) и локальное хранилище браузера.
